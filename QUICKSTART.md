# ⚡ 台股十年資料收集 - 快速開始

## 🎯 30秒快速啟動

```bash
cd taiwan_stock_system
python scripts/collect_all_10years.py --batch-size 5
```

**就這麼簡單！** 系統會自動收集十年完整台股資料。

## 📊 監控收集進度

```bash
# 新開終端視窗
python monitor_collection.py
```

## 🌐 查看分析結果

```bash
# 收集完成後
python run.py
```

訪問：http://localhost:8501
